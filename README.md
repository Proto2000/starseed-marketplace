# Starseed marketplace

A Claude plugin marketplace for [The Starseed Collective](https://starseed-collective.com).

## Plugins

- **[sacred-six](plugins/sacred-six)** — Vanessa's Sacred Six connector + skill:
  your blueprint and her knowledge base, inside your Claude.

## Add this marketplace

**Claude Desktop / claude.ai / Cowork** — Customize → Plugins → Add marketplace →
`Proto2000/starseed-marketplace`, then install **sacred-six**.

**Claude Code**

```
/plugin marketplace add Proto2000/starseed-marketplace
/plugin install sacred-six@starseed
```

The connector signs you in with your Starseed account — nothing per-member or
secret lives in this repo. Setup help: **[starseed-collective.com/learn/connect](https://starseed-collective.com/learn/connect)**.
